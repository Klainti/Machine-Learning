The prepare_dataset.py is a script to get movies from IMDB database by using the OMDB API.
It takes 2 inputs: the number of movies from 2006-2016 and the number of movies from 2017.
The output of the script is a .csv which includes movies (features as actors,writes,directos etc..) from 2006 to 2017.

How to run it ?

0. install python3.6
1. pip3.6 install omdb
2. python3.6 prepare_dataset.py
